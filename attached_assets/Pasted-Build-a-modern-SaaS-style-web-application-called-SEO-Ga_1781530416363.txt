Build a modern SaaS-style web application called SEO Gap Visualizer.

Project Overview

SEO Gap Visualizer helps SEO professionals compare their website against competitor websites to discover content gaps, keyword opportunities, and topic clusters.

The application should look like a professional SEO tool similar to Ahrefs, Semrush, or Surfer SEO.

Use a modern UI with:

Clean dashboard design
Dark/light mode toggle
Responsive layout
Professional charts
Cards and analytics widgets
Sidebar navigation
Modern typography
SEO-themed color palette (blue, purple, white)
